# Käyttöohje

## Asennus

1. Kloonaa projekti:
```
git clone git@github.com:LHuldin/tiralabra.git
```
2. Siirry juurihakemistoon:
```
cd tiralabra

3. Lataa tarvittavat riippuuvuudet:
```
poetry install
```
4. Aktivoi virtuaaliympäristö:
```
poetry shell
```
5. Käynnistä ohjelma:
```
python3 src/main.py
```
